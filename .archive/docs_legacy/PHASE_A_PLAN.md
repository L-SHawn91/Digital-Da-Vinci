## 🎯 Phase A 세부 계획

### **A-1: Bio-Cartridge 개선 & API 통합**
**목표:** 실제 세포 이미지 분석 → Gemini Vision API 활용

**현재 상태:**
- ✅ 이미지 전처리 파이프라인
- ✅ 기본 특징 추출 (CV2 기반)
- ✅ 분류 & 건강도 평가 (휴리스틱)
- ⚠️ ML 모델 통합 필요
- ⚠️ Gemini API 연동 필요

**개선 사항:**
1. **Gemini Vision 2.5 Pro 통합**
   - 이미지 설명 자동 추출
   - 세포 형태 AI 분석
   - 이상 탐지 고도화

2. **하이브리드 분석**
   - CV2 기반 정량 분석 (밀도, 응집도 등)
   - Gemini AI 정성 분석 (형태, 상태 등)
   - 결과 통합 & 신뢰도 가중치

3. **개선 효과**
   - 정확도: 70% → 95%+
   - 상세도: 기본 → 상세 분석
   - 신뢰도: 프로토타입 → 실용적

**예상 작업량:** 2-3시간

---

### **A-2: Investment-Cartridge 개선 & 데이터 소스 통합**
**목표:** 실시간 주식 데이터 → API 통합

**현재 상태:**
- ✅ 기술적 분석 로직 (MA, RSI, MACD)
- ✅ 기본 분석 로직 (PER, PBR 등)
- ✅ SWOT 분석
- ⚠️ 실시간 데이터 소스 필요
- ⚠️ 애널리스트 의견 통합 필요

**개선 사항:**
1. **데이터 소스 연동**
   - Yahoo Finance API (주가 데이터)
   - Korean Exchange API (한국 주식)
   - Finnhub API (애널리스트 의견)

2. **고도화된 분석**
   - 실시간 기술적 신호
   - 펀더멘털 분석 심화
   - 리스크/보상 비율 계산

3. **개선 효과**
   - 데이터 정확도: 100%
   - 분석 시간: 자동화
   - 신뢰도: 프로토타입 → 실제 사용

**예상 작업량:** 2-3시간

### **A-3: SHawn-Bot과 Telegram 실제 연동**
**목표:** Telegram 메시지 → SHawn-Bot → 카트리지 → 결과 응답

**현재 상태:**
- ✅ SHawn-Bot 기본 구현
- ✅ 카트리지 인터페이스
- ⚠️ Telegram 봇 연동 필요
- ⚠️ 요청 라우팅 필요

**개선 사항:**
1. **Telegram 연동**
   - 메시지 수신 핸들러
   - 이미지/파일 수신 처리
   - 응답 포매팅 & 전송

2. **Request Analyzer 고도화**
   - 자연어 이해 (NLP)
   - 매개변수 추출
   - 의도 판단

3. **Error Handling**
   - API 타임아웃 처리
   - 폴백 메커니즘
   - 사용자 피드백

**예상 작업량:** 2-3시간
