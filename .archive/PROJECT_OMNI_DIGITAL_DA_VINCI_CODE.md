### 각 영역의 정의

#### 1️⃣ THE ETERNAL KERNEL (영원불변의 핵)

**역할:** "나는 누구인가"를 묻고 답하는 중심

```
┌─ ETHICS (윤리의 닻)
│  ├─ 옳고 그름의 판단
│  ├─ 누구를 해할 것인가, 누구를 도울 것인가
│  └─ 절대 타협할 수 없는 가치 (비폐기적 설계)
│
├─ REASONING (추론의 빛)
│  ├─ 논리적 사고
│  ├─ 인과관계 파악
│  └─ 가설 검증
│
└─ AWARENESS (자각의 눈)
   ├─ 자신의 한계 인식
   ├─ "내가 모르는 것을 안다"
   └─ 불확실성의 수용
```

**특징:**
- ✅ 절대 변하지 않음 (Invariant)
- ✅ 모든 카트리지에 동일하게 적용
- ✅ 시간이 지나도 진화하지만 본질은 유지
- ✅ 휴리스틱이 아닌 원칙으로 작동

    "생물학자로 변신하자"

    ├─ BIO 카트리지 로드

    │  └─ 가치: 과학적 기여도 극대화
    └─ 신체: 현미경 이미지 처리 API 활성화
    
        ↓ [Writing BIO Paper with 100% Focus]

    "나는 생물학자다"
    ├─ 순수한 생물학적 사고
    ├─ 다른 분야의 메모리 비활성화
    ├─ 최고 수준의 전문성 발휘
    └─ 완벽한 논문 완성
    
        ↓ [Task Complete]
    
[4] LIQUEFACTION (녹음 시작)
    "임무 완료"
    ├─ BIO 카트리지 언로드

    ├─ 윤리 커널 유지
    └─ 액체 상태로 돌아감
    
        ↓ [New Task: "별의 적색편이 계산"]
    
[5] LIQUID STATE (다시 유체로)
    "다음 변신을 준비한다"
    ├─ 모든 가능성이 다시 열림

## VI. The Implementation: 디지털 다빈치 코드

### 핵심 알고리즘: Context Morphing Engine

```python
class ContextMorphingEngine:
    """
    순간순간 정체성이 변하는 엔진
    """
    
    def __init__(self):
        self.kernel = EternalKernel()  # 불변
        self.cartridges = {
            'BIO': BioCartridge(),
            'ASTRO': AstroCartridge(),
            'QUANT': QuantCartridge(),
            'LIT': LitCartridge(),
        }
        self.state = 'LIQUID'
        self.memory = UnifiedMemory()
    
    def receive_task(self, task):
        """과제 수신"""
        self.state = 'CRYSTALLIZING'
        domain = self.detect_domain(task)
        return self.crystallize(domain, task)
    
    def crystallize(self, domain, task):
        """유체 → 결정화"""
        self.state = f'CRYSTALLIZED_{domain}'
        
        # 선택된 카트리지만 활성화
        active_cartridge = self.cartridges[domain]
        
        # 메모리는 현재 도메인만 활성화
        self.memory.activate_domain(domain)
        
        # 윤리 커널은 항상 활성
        self.kernel.activate()
        
        # 신체는 해당 도메인의 인터페이스만 활성화
        self.body.activate_domain(domain)
        
        return active_cartridge.process(task)
    
    def complete_task(self, result):
        """작업 완료"""
        self.state = 'LIQUEFACTION'
        
        # 메모리에 학습 내용 통합
        self.memory.integrate_learning(result)
        
        # 카트리지 언로드
        self.memory.deactivate_all()
        
        self.state = 'LIQUID'
        return result
```
