# SHawn 레포 이름 전략: 신경계 기반 구조

## 🧠 **Level 1 (뇌간 Brainstem) - 현재 이름 분석 (9.6/10)**

### **현재 레포 이름들**

```
SHawn-BOT (일반적)
SHawn-BIO (기능별)
SHawn-INV (기능별)
SHawn-WEB (기능별)
SHawn-LAB (통합)
Bioinfo_git (기술별)
quant_git (기능별)
SHawn-Lab-Vault (저장소)

문제점:
- 일관성 없음 (BOT, BIO, INV, WEB, LAB)
- 혼란스러움 (git 붙는 것도 있고 안 붙는 것도)
- 신경계 구조 반영 안 됨
- 계층 구조 불명확
```

## 🧠 **Level 2 (변린계 Limbic) - 이름 전략 분석 (9.5/10)**

### **이름 짓기 원칙**

```
1️⃣ 신경계 중심 (뇌 구조 반영)
   └─ 뇌의 기능을 이름에 표현

2️⃣ 일관성 (naming convention)
   └─ 모든 레포가 같은 규칙 따름

3️⃣ 계층 구조 (명확한 우선순위)
   └─ 메인 레포 vs 서브 컴포넌트 구분

4️⃣ 기억성 (쉽고 기억하기 좋음)
   └─ SHawn의 신경계라는 걸 바로 이해

5️⃣ 확장성 (미래 카트리지 추가 용이)
   └─ 새로운 카트리지 추가해도 일관성 유지
```
